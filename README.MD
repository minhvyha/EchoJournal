## üåü The "Why"

Most mood tracking apps require you to type out your feelings. While that is great for some, it can be a real barrier during "low energy" mental health days or depressive episodes when even holding a phone to type feels like a lot.

**EchoJournal** is built for accessibility. It lets you speak your mind, transcribes your thoughts in real time, and uses AI to "color" your entry based on your sentiment. No typing, no database, just you and your thoughts in the moment.

## ‚ú® Features

* **Voice-to-Text:** Real-time transcription using the Web Speech API.
* **AI Sentiment Analysis:** Instant analysis of your words via Hugging Face.
* **Dynamic UI:** The background color shifts based on your mood (e.g., cloudy grey for low sentiment to sunny yellow for positive vibes).
* **Privacy First:** No data is stored. This is a fast, ephemeral space for reflection.

## üõ†Ô∏è Tech Stack

* **Frontend:** Next.js + Tailwind CSS
* **AI Logic:** Hugging Face Inference API (implemented directly in the frontend)
* **Transcription:** Web Speech API
* **Deployment:** Vercel

## üöÄ Getting Started

If you want to run this project locally, just follow these steps:

1. **Clone the repo:**
```bash
git clone https://github.com/minhvyha/EchoJournal.git

```


2. **Install dependencies:**
```bash
npm install
# or
pnpm install

```


3. **Run the development server:**
```bash
npm run dev

```


4. **Open your browser:** Head over to [http://localhost:3000](https://www.google.com/search?q=http://localhost:3000) to see it in action!
